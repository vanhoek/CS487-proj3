Project 3

To run all classifiers on dataset:

python3 proj3.py all <dataset>



To run individual classifiers:

python3 proj3.py perceptron <dataset>
python3 proj3.py svm-linear <dataset>
python3 proj3.py svm-rbf <dataset>
python3 proj3.py decision <dataset>
python3 proj3.py knn <dataset>
python3 proj3.py logistic <dataset>


To run with digits from sklearn library, enter 'digits' as the dataset
